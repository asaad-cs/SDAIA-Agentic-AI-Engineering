# SDAIA — Advanced Agentic AI Systems Engineering

> Personal learning repository for the SDAIA 5-day intensive program on Advanced Agentic AI Systems Engineering.

---

## Program Overview

This repository documents everything learned during the **SDAIA Advanced Agentic AI Systems Engineering** program — a 5-day intensive training covering the design, development, and deployment of autonomous AI agents and multi-agent systems.

Each day's folder contains labs, notes, exercises, and a structured summary of topics and takeaways.

---

## Learning Objectives

- Understand the foundations of agentic AI systems and autonomous decision-making
- Design and implement multi-agent workflows and orchestration pipelines
- Apply advanced prompt engineering, tool use, and function-calling patterns
- Build production-ready AI agents using modern frameworks
- Learn evaluation, monitoring, and deployment best practices for agentic systems
